Recessed JetBrains-Mono tech tag — the atomic unit of every stack list and skill card.

```jsx
<div style={{ display: "flex", flexWrap: "wrap", gap: "0.4rem" }}>
  <Chip>Kubernetes</Chip>
  <Chip>Prometheus</Chip>
  <Chip lit>MCP</Chip>
</div>
```

Neutral sunken well by default; `lit` switches to the lime phosphor wash for a featured tag. Always lay several out in a `flex`/`gap` row.
